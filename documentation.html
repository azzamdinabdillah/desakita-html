DesaKita

Comprehensive Village Management System Documentation

Version 1.0.0

Project Overview

DesaKita is a comprehensive village management system designed to streamline administrative tasks, social programs, and community coordination for Indonesian villages. The system provides two main user roles: Village Head and Head of Household, each with specific functionalities and access levels.

Project Setup

Prerequisites

  • npm (comes with Node.js)
  • Git (for version control)

Installation Steps

  1. Clone the repository: 
    git clone [repository-url]
cd desakita-html
  2. Install dependencies: 
    npm install
  3. Start development server: 
    npm run dev
  4. Format code (optional): 
    npm run format
Note: The development server will watch for changes in your CSS files and automatically compile them. The compiled CSS will be available at /src/css/output.css.

Core Components

1. Authentication System

The login page (src/pages/index.html) provides role-based authentication with two user types:

  • Village Head: Full administrative access
  • Head of Household: Limited family-focused access

2. Navigation System

Dynamic sidebar navigation implemented in src/js/sidebar.js with features:

  • Responsive mobile/desktop navigation
  • Active page highlighting
  • Submenu support for complex navigation
  • Path-based menu activation

3. Dashboard Components

Interactive dashboard elements in src/js/dashboard-event-reminder.js:

  • 7-day event calendar
  • Event reminders and notifications
  • Dynamic date navigation
  • Event management interface

4. Form Components

Reusable form elements and utilities:

  • Date Picker: Custom date selection component
  • Image Upload: File upload with preview
  • Modal System: Popup dialogs for forms
  • Tab Navigation: Multi-tab interfaces

User Roles & Permissions

Village Head (Kepala Desa)

Full administrative access with capabilities:

  • Dashboard with village statistics
  • Head of household management
  • Social donation program administration
  • Village profile management
  • Event and development schedule management
  • Data analytics and reporting

Head of Household (Kepala Rumah Tangga)

Family-focused access with capabilities:

  • Family member management
  • Social donation applications
  • Village schedule viewing
  • Personal dashboard
  • Limited administrative functions
